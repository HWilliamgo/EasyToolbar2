# EasyToolbar2
MyEasyToolbar library

### how to use:
1. Add it in your root build.gradle at the end of repositories:
```
	allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
```
2. Add the dependency:
```
	dependencies {
	        compile 'com.github.William619499149:EasyToolbar2:1.0.0'
	}
```
